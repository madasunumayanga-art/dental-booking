# Smile Care Dental — Booking Dashboard

A prototype for an automated dental appointment booking system with WhatsApp simulation.

## Features
- 📋 Live appointment dashboard for clinic staff
- 💬 WhatsApp-style patient booking simulation
- ✅ Real-time booking confirmations
- 📱 Confirm / Done status management

## Run Locally

```bash
npm install
npm run dev
```

## Deploy to Vercel
1. Push this repo to GitHub
2. Go to vercel.com → New Project → Import your repo
3. Click Deploy — done!

## Tech Stack
- React 18
- Vite
- No backend needed for prototype
